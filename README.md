# FlipFlux

Provably-fair coin flip dapp for Sepolia & Abstract testnets.

## Packages
- `hardhat.config.ts` – Solidity contracts, tests, scripts
- `frontend/` (coming next) – React client with wallet integration

## Scripts
- `npx hardhat test`
- `npx hardhat run scripts/deploy.ts --network sepolia`
- `npx hardhat run scripts/deploy.ts --network abstractTestnet`
